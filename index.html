Simpel and Modular Drubuntu is what you need.

Drubuntu basically is a Lamp-Stack that you can use within a virtualization environment to provide a stable and ready to use development platform for Drupal 7 and Drupal 8. The Installation of Drubuntu runs automatically right from the start except from the language selection. With this basic setup and some easy connecting steps you van use your Drupal-Development-Stack with Windows. So If you previously used thinks like Accquias Dev Desktop or even tried to set up Vargant for your own. Give Drubuntu a try. It is as powerful as Vargant and as easy as Acquia's Dev desktop while it serves the same power as a native root server with ease.

As flexible as Drupal is

Drubuntu is not only a virtual root-server for Windows. it is also a fully functional Linux distribution for Drupal developers based on Ubuntu. If you decide to work on an ubuntu driven Desktop environment you can choose forme Desktop environments Additionally we bring Aptna Studio a modern Eclipse based IDE for web developers to work with. Also you can plug-in several features from a basic ftp-server to mor advanced features like git integration for GitHub and Google Code for Aptana Studio just like you need it to improve your workflow.

Drubuntu comes with a pre packed Bootstrap Theme based on Sass and a Gurnt installation so you can start theming for Drupal right after installing Drubunrtu with one of the worlds most popular CSS/JavaScript-Framework.

And The best of all it's absolutely free! Download it now.
